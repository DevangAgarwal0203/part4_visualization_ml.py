# Python Assignment — Part 4: Data Visualization & Machine Learning

## Theme: Student Performance Analysis & Prediction

End-to-end analysis of a student performance dataset — data exploration, visualizations using Matplotlib and Seaborn, and a machine learning classifier to predict pass/fail.

## Tasks

- **Task 1** — Data Exploration with Pandas (5 marks)
- **Task 2** — Data Visualization with Matplotlib — 5 plots (8 marks)
- **Task 3** — Data Visualization with Seaborn — 2 plots (4 marks)
- **Task 4** — Machine Learning with scikit-learn (8 marks + 2 bonus)

## How to Run

Install required libraries:
```
pip install pandas matplotlib seaborn scikit-learn
```

Open the notebook:
```
jupyter notebook part4_visualization_ml.ipynb
```

Then go to **Kernel → Restart & Run All** to execute all cells.
